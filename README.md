## ✨ Öne Çıkan Özellikler

* **Görev Tamamlama:** Onay kutusu ile görevlerin durumunu anlık olarak değiştirme.
* **Veri Kalıcılığı (Local Storage):** Tarayıcının yerel hafızası entegre edilmiştir; sayfa yenilense dahi verileriniz korunur.
* **Modern Tasarım:** Saf CSS ile hazırlanan, mikro etkileşimler ve geçiş animasyonları içeren kullanıcı dostu arayüz.
* **Gelişmiş Mimari:** Proje, veriyi soyutlayan bir "Interface" katmanı üzerinden inşa edilmiştir.

## 🛠️ Kullanılan Teknolojiler

* **Framework:** [ReactJS](https://react.dev/)
* **Dil:** JavaScript
* **Stil:** CSS
* **Yönlendirme:** React Router

## 📁 Proje Yapısı

-   `src/Interfaces/`: Uygulamanın veri katmanı ile haberleşmesini sağlayan **TodoInterface.js** dosyasını barındırır.
-   `src/Components/`: `TodoForm` ve `TodoItem` gibi tekrar kullanılabilir arayüz bileşenlerini içerir.
-   `src/Pages/`: Uygulamanın ana ekranını (`Home.jsx`) ve durum yönetimini barındırır.
-   `src/index.css`: Merkezi tasarım ayarlarını ve animasyon tanımlamalarını içerir.

## 🚀 Kurulum

Projeyi yerel makinenizde çalıştırmak için:

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
2.  Uygulamayı başlatın:
    ```bash
    npm run dev
    ```
